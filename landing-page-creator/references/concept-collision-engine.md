# Concept Collision Engine

Use this before selecting the final creative direction.

The goal is to generate fresh concepts by colliding unrelated but relevant inputs.

## Inputs

Extract:

1. Business object
The real thing the company sells, repairs, transforms, delivers, protects, or manages.

2. Buyer anxiety
What the visitor fears, doubts, hates, misunderstands, or wants to avoid.

3. Proof moment
What would make the visitor trust the company faster: speed, guarantee, local proof, before/after, price clarity, expertise, process, reviews, precision, simplicity.

4. Visual world
Choose an unexpected world that fits the brand but is not obvious:

- cinematic lab
- editorial magazine
- precision instrument
- gallery installation
- control room
- x-ray space
- luxury object reveal
- living map
- mechanical theatre
- floating archive
- liquid material system
- architectural cutaway
- kinetic sculpture
- product ritual
- forensic inspection
- Swiss service counter
- animated blueprint
- glass manufacturing line
- surreal object stage
- tactile macro photography

5. Interaction mechanic
Choose one:

- scroll assembly
- object transformation
- cursor lens
- drag comparison
- pinned story
- magnetic CTA path
- masked reveal
- parallax depth
- timeline scrub
- object orbit
- texture morph
- progressive diagnosis
- light sweep
- zoom tunnel
- section-to-section object continuity

## Required Output

Generate 12 raw concept collisions.

Each concept must follow:

`[Business object] + [buyer anxiety] + [visual world] + [interaction mechanic] + [proof moment]`

Then select 3 concepts worth exploring.

Then select 1 final concept.

Do not choose the safest concept. Choose the concept with the strongest mix of:

- wow factor
- business specificity
- conversion clarity
- implementation feasibility
- mobile viability

## Rejection Rules

Reject concepts that:

- sound like a generic agency website
- are only a layout idea
- rely only on gradients/cards
- could fit another random business
- do not create a memorable first viewport
- cannot be explained visually in 3 seconds
