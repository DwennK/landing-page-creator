# Bad Output Repair Loop

If the result looks AI-generated, do not polish it. Re-concept it.

## Diagnosis

Score the current output from 1 to 5:

- Hero wow factor
- Client specificity
- Visual originality
- Scroll story
- Component usefulness
- Typography/art direction
- Mobile impact
- Sellable demo feeling

Any score under 4 must be fixed before delivery.

## Fix Order

1. Replace the hero concept.
2. Rebuild the main visual scene.
3. Add a meaningful scroll transformation.
4. Replace generic cards with proof, object, comparison, or interaction.
5. Improve typography and spacing.
6. Re-check mobile.
7. Delete any section that still feels like filler.

## Forbidden Fixes

Do not fix a weak design by:

- adding more gradients
- adding more shadows
- adding more cards
- adding more icons
- adding fake metrics
- adding decorative motion
- adding another generic section

If the concept is weak, replace the concept.
