# ImageGen Prompt System

## Prompt Template

Create a premium visual asset for a landing page.

Subject: [main object / metaphor]
Composition: [layout, focal point, negative space]
Lighting: [soft studio, cinematic rim light, glass reflections, etc.]
Material/style: [metal, glass, liquid, paper, architectural, product photography]
Mood: [calm, premium, technical, trustworthy, futuristic, human]
Camera: [macro, isometric, wide-angle, top-down, editorial]
Brand fit: [why it matches the company]
Usage: [hero background, section visual, CTA backdrop, icon style]
Negative constraints: no text, no logos, no stock-photo people, no messy details, no random holograms, no cliché AI gradients, no distorted hands, no illegible UI.

## Example — Phone Repair Hero

Create a premium cinematic hero visual for a smartphone repair and resale shop. A sleek modern smartphone floats in a dark precision repair lab, partially opened in an elegant exploded-view composition, with tiny glass, metal, camera, battery, and circuit layers suspended with perfect spacing. Soft blue-white rim lighting, subtle reflections, premium black and graphite materials, Swiss clean precision mood, shallow depth of field, dramatic negative space on the left for headline text. No readable text, no logos, no hands, no fake UI, no messy cables, no stock-photo look, no neon cyberpunk excess.
